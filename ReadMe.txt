Image to Chord 1.0.0

The program can convert an image to a 5 note chord.

Instructions:

- Drag and drop an image file onto the program icon.
- A terminal window will open and display the corresponding
  chord and the matching scales.
- You can alternatively start the program in a terminal window,
  using the image filename as an argument.

A color in RGB format, resulting from converting the image to a color,
will also be displayed on top of the chord.
